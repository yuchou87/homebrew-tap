# homebrew-tap

[Homebrew](https://brew.sh) tap for [@yuchou87](https://github.com/yuchou87)'s tools.

## Usage

```bash
# install directly
brew install yuchou87/tap/<formula>

# or tap once, then install by name
brew tap yuchou87/tap
brew install <formula>
```

Upgrade with `brew upgrade <formula>`; uninstall with `brew uninstall <formula>`.

## Formulae

| Formula | Description |
|---|---|
| [tokenpulse](https://github.com/yuchou87/tokenpulse) | Keep your Mac awake and show Claude Code usage on an ESP32-S3 desk gadget |

_More tools will be added here over time._
